# Simple-Go-Service

To start the service, run: 

`go run cmd/simple-service/main.go`

## Endpoints
<hr>

GET: `http://localhost:1323/users`

POST: `http://localhost:1323/users/:id`
Body: `{"name":"`[name]`"}`

PUT: `http://localhost:1323/users/:id`
Body: `{"name":"`[name]`"}`

DELETE: `http://localhost:1323/users/:id`
